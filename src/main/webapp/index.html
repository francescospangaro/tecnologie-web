In these examples we will see how PROCESS and VALIDATE different inputs.

QUERY STRING

List of courses: In this example you will see a list of courses and when selecting one, we will see the details

FORMS

Form with text input: In this example you will see how to parse text input
Form with checkbox input: In this example you will see how to parse checkbox input
Form with radio button input: In this example you will see how to parse radio button input
Form with select: In this example you will see how to parse input in select format by simulating registration to course
Upload File: In this example you will see a simple workflow to upload a file

Note 1: in these examples to test the corner cases we did not use the HTML tag "REQUIRED", but it should be added when possible. As well as other tags for validation (e.g. min and max for number input)

Note 2: in the examples to list and add courses, the class CourseManager acts as a "mock-up" of the the database. As you advance in the course you will learn to create classes to write and read from the database and that will be the standard for the course.
